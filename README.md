# Hacker-News
Simple SPA with [Hacker News Api]()

## Installation

**1. Download the project**

```bash
git clone https://github.com/Dmitriymgtu/Hacker-News.git
```
**2. Go to the application folder**

**3. Run application**

```bash
npm start
```

or

```bash
yarn start
```
